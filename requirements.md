<p class="message">
	TeamPass needs to be installed on a web server.
</p>

### Server Requirements

* Apache v2.0 (and after)
* MySQL v5.1 (and after)
* PHP v5.4.0 (and after) since release 2.1.23

### PHP enabled modules

* mcrypt
* mbstring
* openssl
* LDAP if using LDAP identification
